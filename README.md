# Kutabaru Web Agency - Local SEO Engine

![Astro](https://img.shields.io/badge/Main%20Framework-Astro-ff5d01?style=for-the-badge&logo=astro)
![Tailwind CSS](https://img.shields.io/badge/Styling-Tailwind%20CSS-38bdf8?style=for-the-badge&logo=tailwind-css)
![SEO](https://img.shields.io/badge/Strategy-Local%20SEO%20pSEO-success?style=for-the-badge)

Website agensi jasa pembuatan website **Kutabaru** yang dirancang dengan arsitektur modern untuk mendominasi peringkat mesin pencari di tingkat lokal menggunakan strategi *Programmatic SEO* (pSEO).

## 🚀 Fitur Utama

- **Zero-JS Static Site:** Dibangun dengan **Astro** untuk performa Core Web Vitals yang sempurna.
- **Parent-Child Branding:** Struktur branding pusat (Kutabaru) dengan puluhan landing page lokal yang otomatis ter-generate.
- **Local Business Intelligence:** Implementasi Schema.org JSON-LD yang dinamis untuk relevansi lokasi yang tinggi di mata Google.
- **Contextual Lead Capture:** Integrasi WhatsApp dengan parameter pesan otomatis berdasarkan wilayah asal pengunjung.

## 🛠️ Tech Stack

- **Framework:** Astro (SSG)
- **UI & Styling:** Tailwind CSS
- **Data Management:** Astro Content Collections (Type-safe JSON/Markdown)
- **Language:** TypeScript

# Brainstorming Skill

Gunakan skill ini untuk memicu diskusi kreatif dan teknis sebelum mengeksekusi rencana.

## Instruksi
- Berikan minimal 3 opsi berbeda untuk setiap masalah besar.
- Sertakan pro/kontra dari sisi teknis (performa, maintainability) dan user (UX, estetika).
- Rekomendasikan opsi terbaik berdasarkan `docs/technical-design.md`.
